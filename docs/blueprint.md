# **App Name**: KarmaSphere

## Core Features:

- Real-time Project Chat: Implement a Socket.io-powered chat room for each project, allowing team members to communicate in real-time.
- Message Storage: Store chat messages in a MongoDB database for persistent history and retrieval.
- Task Assignment Notification: Notify team members in real-time when a task is assigned to them using Socket.io notifications.
- Task Status Update Notification: Notify project managers in real-time when a task's status is updated, using Socket.io notifications.
- User Preferences for Notifications: Allow users to customize notification preferences (e.g., in-app, email) to suit their individual needs.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to convey trust and intelligence.
- Background color: Light gray (#F0F0F0) for a clean, professional look.
- Accent color: Teal (#008080) to draw attention to important notifications and actions.
- Body and headline font: 'PT Sans' for a modern, readable feel.
- Use simple, clear icons to represent task status and notification types.
- Use a clean, card-based layout to display project chats and notifications, promoting clear visual hierarchy.
- Implement subtle animations for new messages and notifications to provide visual feedback.
